# The `math` Module

Reference: https://docs.python.org/3/library/math.html.

The `math` module contains helpful numeric functions:

```python
import math # load the module to avoid `NameError: name 'math' is not defined`

math.pi #> 3.141592653589793

math.sqrt(4) #> 2.0

math.ceil(4.4) #> 5.0

math.floor(4.4) #> 4.0

math.trunc(4.4) #> 4
```
